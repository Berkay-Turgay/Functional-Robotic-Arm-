## Functional-Robotic-Arm
This project showcases a remotely controlled robotic arm built with Robotis STEM structural parts. It is powered by Dynamixel AX-12A servos and controlled wirelessly via a Robotis CM530 controller board.

# Project Development Environment
This project was developed using RoboPlus 1.1.3 editor, source code was written **RoboPlus Task** and motion configuration is done **RoboPlus Motion**

# Feature
- CM530 Controller
- Dynamixel AX-12A Servo Motor
- RC-100B Remote Controller
- IR-10 Robotis IR Receiver Module
- Robotis STEM Connection Parts ve Cables
- LB-010 LIPO Battery 11.1V 1000mAh
